# Калькулятор психоматрицы

Веб-приложение для расчёта психоматрицы по дате рождения на основе метода статистического психоанализа О. Перцевой. Позволяет получить «цифровой» психологический портрет человека без регистрации и установки.

## Демо

Открыть `index.html` в любом современном браузере — серверная часть не требуется.

## Функциональность

- Ввод даты рождения в формате `дд.мм.гггг` с автоматическим форматированием
- Расчёт психоматрицы по 14 параметрам:
  - Число судьбы, Темперамент, Характер
  - Здоровье, Удача, Цель, Энергия
  - Логика, Долг, Семья, Интерес
  - Труд, Память, Привычки, Быт
- Отображение дополнительных чисел
- Копирование результата в буфер обмена
- Кнопка заказа персонального разбора через WhatsApp
- Информационная панель об эксперте и услугах (модальное окно на мобильных, боковая колонка на десктопе)
- Адаптивная вёрстка (mobile-first)

## Технологии

| Технология | Назначение |
|---|---|
| HTML5 | Разметка |
| CSS3 (Custom Properties) | Стилизация и адаптив |
| Vanilla JavaScript | Вся логика на стороне клиента |

Фреймворки и сборщики не используются. Приложение работает как статическая страница.

## Запуск

```bash
# Клонировать репозиторий
git clone https://github.com/maxim-semikov/calc-matrix.git
cd calc-matrix

# Открыть в браузере
open index.html
```

Либо запустить через любой статический сервер, например:

```bash
npx serve .
```
